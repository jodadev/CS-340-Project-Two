# CS340 Project Two Reflection

## Maintainable, Readable, and Adaptable Code
Writing maintainable, readable, and adaptable programs starts with separating responsibilities and keeping code organized. In this project, the CRUD Python module from Project One played a key role in achieving this. By isolating all database operations into a single module, the dashboard code remained clean and focused only on user interaction and visualization.

This approach made the code easier to read and maintain because changes to database logic could be made in one place without affecting the rest of the application. It also made the project more adaptable, since the same CRUD module could be reused in other applications that require interaction with the same database structure.

In the future, this CRUD module could be reused for other dashboards, data analysis tools, or backend services that need to interact with MongoDB. It provides a reusable foundation that reduces duplication and improves consistency across projects.

## Problem-Solving Approach as a Computer Scientist

When approaching this project, I focused on breaking the problem into smaller components: database access, filtering logic, user interface, and interaction between components. Instead of trying to build everything at once, I developed each part step-by-step and verified functionality along the way.

This project differed from previous assignments because it required integrating multiple systems, including a database, backend logic, and a front-end dashboard. It was not just about writing code, but about designing how different parts of a system communicate and work together.

In the future, I would use a similar structured approach when designing databases and applications. This includes understanding the client requirements first, designing the data model, building reusable components, and testing each part incrementally to ensure reliability.

## The Role of Computer Scientists and Why It Matters

Computer scientists design and build systems that allow organizations to use data effectively. Their work transforms raw data into meaningful insights and tools that improve decision-making.

In this project, the dashboard enables Grazioso Salvare to quickly identify dogs that meet specific rescue training criteria. Instead of manually reviewing large datasets, users can filter and visualize the data in real time, making the process faster and more efficient.

This type of work matters because it directly improves how organizations operate. By creating tools that simplify data analysis and reduce manual effort, computer scientists help businesses save time, reduce errors, and make better-informed decisions.
